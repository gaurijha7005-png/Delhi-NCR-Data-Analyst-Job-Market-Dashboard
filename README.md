# 📊 Delhi/NCR Data Analyst Job Market Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard that analyzes Data Analyst job postings across the Delhi/NCR region.

The dashboard helps identify:

- Hiring trends
- Most active hiring companies
- Salary insights
- Remote vs Hybrid vs On-site jobs
- Most in-demand technical skills
- Skill gaps
- Experience-wise opportunities
- Geographic distribution of jobs

---

## Project Objectives

- Analyze hiring demand in Delhi/NCR
- Identify top hiring locations
- Compare work modes
- Discover the most demanded technical skills
- Analyze experience-level requirements
- Visualize hiring distribution using maps
